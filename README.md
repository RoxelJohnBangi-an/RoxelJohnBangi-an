## 🐾 My Capstone Project – DailyFur
**DailyFur** is an Android mobile app that helps pet owners:

✔ Schedule vet appointments with notification badges  
✔ Chat with veterinary clinics  
✔ Locate nearby vet clinics via geo-mapping  
✔ Teleconsult with veterinarians  
✔ Emergency pet service coordination  
✔ Clinic ratings & in-app pet store  

Built using **Kotlin + Firebase + Android Studio**
